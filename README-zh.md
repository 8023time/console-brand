# Console-Brand 在线版

一个在线平台，用于在浏览器控制台快速生成个性化、醒目的日志输出。无需安装或配置，通过可视化操作即可定制样式、颜色、Badge、渐变、ASCII 或 Emoji，实时预览效果并复制使用。

## 核心功能

- **在线个性化定制**  
  通过可视化面板调整文本、Badge、渐变、颜色、字体、阴影、圆角等样式，实时预览效果。

- **丰富输出形式**  
  - Badge（标签 | 值）  
  - 渐变高亮文本  
  - ASCII 艺术和 Logo  
  - 大尺寸 Emoji  
  - 对象 / JSON / 表格打印

- **快速预设**  
  内置成功、错误、警告、霓虹等常用样式，一键生成高辨识度控制台日志。

- **复制即用**  
  在线生成代码后，可直接复制到项目中使用，无需额外依赖。

---

## 使用场景

- 在线生成控制台品牌输出  
- 项目调试信息高亮展示  
- 开源项目或内部工具的醒目日志提示  
- 展示 Logo、ASCII 艺术或 Emoji 品牌标识  

---

## 技术特点

- **纯前端在线实现**，无需安装  
- **可视化操作**，即改即用  
- **TypeScript 与现代浏览器兼容**  
- **支持渐变、阴影、字体、圆角等高级样式**
